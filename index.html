This document contains text declaring the levels for the Slider game

Before publishing, check that capital r does not exist in a level

Each level is represented by a sequence of characters and new lines like follows:

r r r r r r r r r r
x r r r r r r r r r
x x r r r r r r r r
x x x r r r l r r r
x x x x r r l r r r

Meanings of characters

r - completely random block
f - A random straight track or curved track
h - a house
t - a tree
b - a blank block
l - a straight block facing forwards
s - a straight block facing sideways
bl - curve bottom to left
tl - curve top to left
tr - curve from top to right
br - curve from bottom to right
x - no block

Capital letter - a block with a coin

If the car is to start on the block, the block character will be followed by a 'c' like so:
lc

Note that the top line of the sequence of characters will appear in the front of the screen when the map is generated

The starting positon of the camera is declared below the level declearation as follows (rotation is in degrees):
cameraPosition -5 5 -5
cameraRotation -30 0 0 

The starting rotation of the camera can be declared as follows:
carRotation 0 0 0

the speed is declared below as follows:
speed .1

Each level starts with the text "Level [level number]" and ends with "Level complete"
